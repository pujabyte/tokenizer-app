# Tokenizer App — Clone tokenisasi.io

Next.js replica of tokenisasi.io dashboard with identical design.

## Tech Stack
- **Next.js** 16.2.6 (App Router)
- **React** 19.2.4
- **styled-components** 6.4.1
- **Tailwind CSS** 4
- **Radix UI** (Dialog, Tabs, Switch, Tooltip, DropdownMenu)
- **Lucide React** (icons)
- **Lodash** 4.17.21
- **clsx** + **class-variance-authority**

## Run
```bash
npm install
npm run dev
```
